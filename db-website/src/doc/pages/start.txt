====== Bytle Db - Documentation ======


===== About =====
''Bytle-Db'' is a command line client called [[db|db]] that helps you perform data operations in a simple and effective way.

<WRAP round tip>
If you have no previous experience with ''Bytle-Db'', we advice you to follow this [[getting_started|getting started]].
</WRAP>

===== Data Structure =====

[[..:db|Bytle Db]] helps you operate on the following data structure or definition:
  * [[table:table|Table]] - a relational table supporting its operations through Sql.
  * [[schema:start|Schema]] - a table namespace (where the tables are stored)
  * [[fkey:start|Foreign Key]] - relationships between tables
  * [[query:start|Query]] - a Sql query (ie a derived table)
  * [[csv:csv|Csv]] - a Csv table
  * [[excel:excel|Excel]] - a Excel table
