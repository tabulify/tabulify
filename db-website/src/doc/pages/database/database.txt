====== Bytle Db - Database ======


===== About =====
For ''Bytle Db'', a ''database'' is a data store from where you can retrieve data.

===== Properties =====
==== Name ====
For ''Bytle Db'', the name of a database is its unique identifier and therefore should be unique inside a [[store|database store]].

The name of a database is used as the first part of [[..:data_uri|data structure URI]].
