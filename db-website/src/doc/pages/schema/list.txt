====== Bytle Db - Schema List ======


===== About =====
''schema list'' will list the tables present in a [[start|schema]]


===== Syntax =====
<unit>
<code dos>
db schema list --help
</code>
<console>
List schemas

Syntax:

	db schema list <schemaName|pattern...>

where:

	Args:
		* schemaName|pattern...      Names of a schema or a glob patterns
</console>
</unit>

===== Example =====


You can list the schema with the following command:
<unit>
<code dos>
db schema list
</code>
<wrap important>Because Sqlite is the default database and is not supporting any schema, there is no schemas found.</wrap>
<console>
No schemas found
</console>
</unit>
