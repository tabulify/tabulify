====== Bytle Db - Schema ======

===== About =====
A ''relational schema'' is a [[..:resource|data resource]] of a [[relation:relation|relational database]]



===== URI =====
The [[..:data_uri|Data URI]] that defines the location of a schema has the following form:
<code>
@database/schema
</code>
where:
  * ''database'' is the [[database:database|database name]]
  * ''schema'' is the schema name
