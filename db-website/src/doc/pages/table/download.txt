====== Bytle Db - Download a table ======


===== About ===== 
The ''table download'' action permits you to download a [[table|relational table]] into a [[:tabular_file|file]]. 

The target data file may be:
  * a [[csv:csv|CSV]]
  * or [[excel:excel|Excel file]].



===== Syntax =====
<unit>
<code dos>
db table download --help
</code>
<console>
Download a table into a file.

Syntax:

	db table download [options|flags] <TableName>

where:

	Local
		* -o,--output <path>         defines the directory of the downloaded file (Default: currentWorkingDirectory)
		* --sd <value>               defines the SOURCE driver for the SOURCE database (Example: oracle.jdbc.OracleDriver)

	Local
		* --cif <value>              If present, the values of clob columns will be written in a apart file

	Args:
		* TableName                  No Description available


Example:
download-tdf downloaded.csv toDownload.sql
</console>
</unit>
