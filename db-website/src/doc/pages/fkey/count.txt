====== Bytle Db - Foreign Key - Count ======


===== About =====
''count'' is an action of the [[fkey|foreign key module]] that counts the number of [[fkey|foreign keys]].


===== Syntax =====


<unit name="Syntax">
<code dos>
db fkey count --help 
</code>
<console>
22:37:08 - INFO - (conf) word was not found
22:37:08 - INFO - (help) word was found with the value: true

Count links (foreign keys)

Syntax:

	db fkey count [options|flags] <tableUri...>

where:

	Local
		* --store <value>            The path where the database information are stored

	Args:
		* tableUri...                One or more name table uri (ie @database[/schema]/table)
</console>
</unit>
where:
  * ''TableUri'' is a [[table:table#uri|Table Uri]]

===== Example =====

<unit>
  * Creating [[..:sample:tpcds|the tpcds sample schema]] to create [[fkey|foreign keys]]
<code dos>
db sample create --no-strict tpcds-dwh @sqlite
</code>
</unit>
  * Counting all foreign keys
<unit>
<code dos>
db fkey count @sqlite/*
</code>
<console>
82 ForeignKeys

22:37:21 - INFO - Bye !
</console>
</unit>

  * Counting all foreign keys for the table ''catalog_returns''
<unit>
<code dos>
db fkey count @sqlite/catalog_returns
</code>
<console>
13 ForeignKeys
</console>
</unit>
