====== Bytle Db - Foreign Key - List Operation ======


===== About =====
''fkey list'' is an operation of the [[start|foreign key module]] that list the [[fkey|foreign keys]].


===== Syntax =====
<unit name="Syntax">
<code dos>
db fkey list --help 
</code>
<console>
09:23:11 - INFO - (conf) word was not found
09:23:11 - INFO - (help) word was found with the value: true

List foreign keys

Syntax:

	db fkey list [options|flags] <TableUri...>

where:

	Local
		* --store <value>            The path where the database information are stored

	Local
		* --c                        Show also the columns if present

	Args:
		* TableUri...                One or more name table uri (ie @database[/schema]/table)
</console>
</unit>
where:
  * ''TableUri'' is a [[table:table#uri|Table Uri]]

===== Example =====
<unit>
  * Creating [[..:sample:tpcds|the tpcds sample schema]] that have ''foreign keys''
<code dos>
db sample create --no-strict tpcds-dwh @sqlite
</code>
</unit>
