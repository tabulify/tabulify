# Data Def file



## Metadata Vocabulary for Tabular Data

https://www.w3.org/TR/2015/REC-tabular-metadata-20151217/
  
A [metadata document](https://www.w3.org/TR/2015/REC-tabular-metadata-20151217/#metadata-format)
is a JSON document which holds an object at the top level. 

This object is a description object of either:
  * a group of tables 
  * or a single table.
