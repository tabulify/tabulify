====== Bytle Db - Relational Database ======


===== About =====
[[db|Bytle db]] supports the management of relation schema with the help of the following [[module|modules]]:
  * [[table:table]] - to perform operation on a relational table
  * [[schema:schema]] - to perform operation on a relational schema
  * [[database:database]] - to add/create/update database connections parameters
