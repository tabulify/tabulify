====== Bytle Db - Foreign Key Module (fkey) ======

===== About =====
''fkey'' is a [[:module|module]] that let you perform operation on foreign key (ie to see the relation between tables)


===== Syntax =====
<unit name="Syntax">
<code dos>
db fkey --help
</code>
<console>
13:22:51 - INFO - (conf) word was not found
13:22:51 - INFO - (help) word was found with the value: true

operations on foreign key (ie on relation and constraints between tables)

Syntax:

	db fkey <command>

where:

	Command (is one of):
		* list                          list the links between tables (foreign keys)
		* count                         count the total number of links
</console>
</unit>
where:
  * [[list]]
  * [[count]]
