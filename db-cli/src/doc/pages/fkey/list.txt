====== Bytle Db - Foreign Key List Operation ======


===== About =====
''fkey list'' is an operation of the [[fkey|foreign key]] module that list the foreign keys.


===== Syntax =====
<unit name="Syntax">
<code dos>
db fkey list --help
</code>
<console>
21:15:32 - INFO - (conf) word was not found
21:15:32 - INFO - (help) word was found with the value: true

List foreign keys

Syntax:

	db fkey list [options|flags] <tableName|pattern...>

where:

	Local
		* --c                        Show also the columns if present

	Args:
		* tableName|pattern...       Names of table or glob patterns
</console>
</unit>

===== Example =====
<unit>
  * [[..:sample:create|Creating a sample schema]] that have ''foreign keys''
<code dos>
db sample create --no-strict tpcds-dwh @sqlite
</code>
</unit>
  * Listing the foreign keys
<unit>
<code dos>
db fkey list @sqlite
</code>
<console>
No foreign key found
</console>
</unit>
