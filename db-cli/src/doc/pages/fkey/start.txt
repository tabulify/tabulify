====== Bytle Db - Foreign Key Module (fkey) ======

===== About =====
''fkey'' is a [[:module|module]] that let you perform operation on [[fkey|foreign key]] (ie to see the relation between tables)

<WRAP round important>
The action are more on relationship than on foreign key.

Ie when you list the foreign key for a primary table, you get also the foreign key towards its primary key
</WRAP>

===== Syntax =====
<unit name="Syntax">
<code dos>
db fkey --help
</code>
<console>
operations on foreign key (ie on relation and constraints between tables)

Syntax:

	db fkey <command>

where:

	Command (is one of):
		* list                       list the links between tables (foreign keys)
		* count                      count the total number of links (foreign keys)
</console>
</unit>
where:
  * [[list|list]]
  * [[count|count]]
