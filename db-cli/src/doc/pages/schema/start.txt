====== Bytle Db - Schema ======


===== About =====
''schema'' is a [[:module|module]] to perform operations on all [[table:table|table]] of a [[schema|relational database schema]] at once.

<wrap tip>Db can also create sample schema. See [[sample:sample]]</wrap>


===== Syntax =====
<unit>
<code dos>
db schema --help
</code>
<console>
operations on a schema

Syntax:

	db schema <command>

where:

	Command (is one of):
		* fill                          fill a schema with generated data
		* list                          list schemas
</console>
</unit>
where the submodule are:
  * [[list]]
  * [[fill]]
