====== Bytle Db - Documentation ======


===== About =====
''Bytle-Db'' is a command line client called [[db|db]] that helps you perform data operations in a simple and effective way.

<WRAP round tip>
If you have no previous experience with ''Bytle-Db'', we advice you to follow this [[getting_started|getting started]].
</WRAP>

===== Data Structure =====
For ''Bytle Db'', a [[database:database|database]] is an application that stores data. 

[[..:db|Bytle Db]] helps you operate on the following data structure:
  * [[table:|Table]] - a relational table supporting its operations through SQL.
  * [[schema:schema|Schema]] - a table namespace (where the tables are stored)
  * [[fkey:fkey|Foreign Key]] - relations between tables
  * [[query:query|Query]] - a SQL query (ie a derived table)
  * [[csv:start|Csv]] - a CSV table
  * [[excel:start|Excel]] - a Excel table
