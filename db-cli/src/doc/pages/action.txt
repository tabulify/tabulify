====== Bytle Db - Action ======


===== About =====
An action is an operation that is performed on a [[module|module]]. This is generally the second word found just after the [[module|module name]]

Action parameters may be given through the use of [[option|options]].

**Syntax**

<code dos>
db module action [options|flag] argument
</code>

where:
  * [[db]]
  * [[module]]
  * [[option]]
  * [[flag]]
