====== Query - Download ======


===== About =====
The ''download'' action of the [[query|query]] module permits to download the data on your local file system

===== Syntax =====

<unit>
<code dos>
db query download --help
</code>
<console>
download a query into a file

Syntax:

	db query download [options|flags] <(Query|File.sql)>

where:

	Local
		* -o,--output <path>            defines the path of the output path (Ex: output/query.csv)

	Local
		* -c,--clob <value>             If present, the values of clob columns will be written in a apart file

	Args:
		* (Query|File.sql)              No Description available


Example:
download--output QueryDownloaded.csv QueryToDownload.sql
</console>
</unit>
