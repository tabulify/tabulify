====== Bytle Db - Relational Database ======


===== About =====
[[:db|Bytle db]] supports the management of [[wp>Relation_(database)|relational data structure]] with the help of the following [[:module|modules]]:
  * [[database:start]] - to add/create/update database connections parameters
  * [[query:query]] - to perform operation on a SQL relational query
  * [[table:start]] - to perform operation on a relational table
  * [[schema:start]] - to perform operation on a relational schema
  * [[fkey:fkey]] - to manage foreign keys (ie relation between tables)


===== Bonus =====
The [[sample:schema|sample module]]  contains several sample relational schema.

===== Supported Relational Database =====

All relational database that provides a mature JDBC driver and supports [[wp>SQL#Interoperability_and_standardization|ANSI SQL]] for the creation of data structure should work out of the box.

For the supported database, ''Bylte Db'' introduces a layer that:
  * corrects the driver mistake
  * and/or implements the database SQL dialect
