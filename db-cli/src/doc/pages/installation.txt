====== Bytle Db - Installation ======


===== About =====
This pages gives you all the step to install successfully [[db|Bytle Db]]

===== Steps =====
  * Download the zipped installation file 
  * Create a directory called the [[bytle_db_home|Bytle Db Home]]
<code dos>
mkdir c:\bytle-db
</code>
  * Unzip the downloaded file into the [[bytle_db_home|Bytle Db Home]]
<code dos>
unzip btyle-db-cli-1.1.0-SNAPSHOT-windows-x64.zip -d c:\bytle-db
</code>
  * Optional, create a symlink into your bin directory to be able to call [[db|Bytle Db]] from everywhere.
<code dos>
mklink c:\bytle-db\db.cmd %USERPROFILE%\bin\db.cmd
</code>
