====== Bytle Db - Database ======


===== About =====
''Database'' is a [[..:module|Bytle-Db module]] that helps you manage connections to database.


===== Syntax =====
<unit>
<code dos>
db database --help
</code>
<console>
06:05:44 - INFO - (conf) word was not found
06:05:44 - INFO - (help) word was found with the value: true

Database management

Syntax:

	db database <command>

where:

	Command (is one of):
		* add                           add a database
		* list                          list the databases
		* info                          show database information
</console>
</unit>
