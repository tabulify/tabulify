====== Bytle Db - Database Module ======


===== About =====
''Database'' is a [[..:module|Bytle-Db module]] that helps you manage [[database|databases]] (mostly database connections informations).

Within the database module, you can:
  * [[list]] - list the databases in a [[store|databases store]] in a table format
  * [[info]] - get information on a database in a forms format.
  * [[add]] - create/add a database in a [[store|databases store]] 
  * [[remove]] - remove a database from a [[store|databases store]] 
  * [[upsert]] - update or create a database in a [[store|database store]] 



===== Syntax =====
<unit>
<code dos>
db database --help 
</code>
<console>
Database management

Syntax:

	db database <command>

where:

	Command (is one of):
		* add                           add a database
		* upsert                        update or add a database if it does't exist
		* list                          list the databases
		* info                          show database information
		* remove                        Remove a database
</console>
</unit>
