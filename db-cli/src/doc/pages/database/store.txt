====== Bytle Db - Databases Store ======


===== About =====
The ''database store'' is a local file where the [[database|databases information]] are stored.


===== Location =====

In every [[database#action|database action]], you can specify the location of this file with the ''store'' option.


Or globally if you set the path by creating the following environment variable ''BYTLE_DB_DATABASES_STORE''.


Default:
  * Windows: ''C:\Users\userName\AppData\Local\db\databases.ini''
  * Linux: ''~/.db/databases.ini''

===== Format =====
This file follows the same semantic than ODBC.ini file where each section is a database (ie DSN).


===== Password storage =====
Every password in a database store is encrypted.

The encryption ciphers used are approved by the Computer security division of the USA department of commerce via the National Institute of Standard and Technology) and follows the [[https://csrc.nist.gov/publications/detail/sp/800-38a/final|SP800-38A recommendations]].

When a database connection need a password, a ''passphrase'' is mandatory. This passphrase should be unique by databases store file.
