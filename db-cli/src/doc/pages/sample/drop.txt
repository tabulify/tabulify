====== Bytle Db - Drop a sample schema ======


===== About =====
''sample drop'' is an action of the the [[schema|sample module]]. It will drop every tables that belongs to a [[schema|sample schema]].


===== Syntax =====

<unit>
<code dos>
db sample drop --help 
</code>
<console>
Drop the tables that belongs to a sample database schema.

Syntax:

	db sample drop [options|flags] <SampleSchemaName> <SchemaUri>

where:

	Local
		* --store <value>            The path where the database information are stored

	Local
		* --no-strict                if set, it will not throw an error if a table is not found

	Args:
		* SampleSchemaName           The name of the sample schema. One of tpcds, tpcds-dwh, tpcds-stg, tpcds-store-sales, BytleSchema
		* SchemaUri                  A relational schema uri (ie @database[/schema]
</console>
</unit>

===== Example =====
  * Creating the [[tpcds|tpcds store schema]] in the [[database:default|default sqlite database]].
<unit>
<code dos>
db sample create --no-strict tpcds-store-sales @sqlite
</code>
</unit>
  * Dropping it
<unit>
<code dos>
db sample drop tpcds-store-sales @sqlite
</code>
