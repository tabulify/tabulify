====== Bytle Db - Create a sample schema ======


===== About =====
''sample create'' is an operation of the [[schema|sample module]] where you can create a [[schema|sample schema]] (ie a list of [[table:table|tables]]).



===== Syntax =====
<unit name="Syntax">
<code dos>
db sample create --help 
</code>
<console>
11:06:19 - INFO - (conf) word was not found
11:06:19 - INFO - (help) word was found with the value: true

Create sample database schema.

Syntax:

	db sample create [options|flags] <SampleSchemaName> <SchemaUri>

where:

	Local
		* --no-strict                if set, it will not throw an error if a table already exist

	Local
		* --store <value>            The path where the database information are stored

	Args:
		* SampleSchemaName           The name of the sample schema. One of tpcds, tpcds-dwh, tpcds-stg, tpcds-store-sales, BytleSchema
		* SchemaUri                  A relational schema uri (ie @database[/schema])
</console>
</unit>

===== Example =====
  * [[drop|Drop]] the [[tpcds|tpcds store sales schema]] from the [[database:default|default sqlite database]] (to be sure that it doesn't exist)
<unit>
<code dos>
db sample drop --no-strict tpcds-store-sales @sqlite
</code>
</unit>
  * Create it
<unit>
<code dos>
db sample create tpcds-store-sales @sqlite
</code>
</unit>
