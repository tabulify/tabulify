====== Bytle Db - Fill a sample schema ======


===== About =====
The ''sample fill'' is an action of the [[schema|sample module]]


===== Syntax =====
<unit>
<code dos>
db sample fill --help
</code>
<console>
07:59:36 - INFO - (conf) word was not found
07:59:36 - INFO - (help) word was found with the value: true

(Create|Load) data into a sample database schema.

Syntax:

	db sample fill [options|flags] <SampleSchemaName> <SchemaUri>

where:

	Local
		* --store <value>            The path where the database information are stored

	Local
		* -s,--scale <value>         The size of the generated data in Gb (works only for tpc schema)

	Args:
		* SampleSchemaName           The name of the sample schema. One of tpcds, tpcds-dwh, tpcds-stg, tpcds-store-sales, BytleSchema
		* SchemaUri                  A relational schema uri (ie @database[/schema]
</console>
</unit>

===== Example =====
  * Create the [[tpcds|datawarehouse TPC-DS schema]].
<unit>
<code dos>
db sample create --no-strict tpcds-store-sales @sqlite
</code>
</unit>
<unit>
  * Fill it with data.
<code dos>
db sample fill tpcds-store-sales @sqlite
</code>
