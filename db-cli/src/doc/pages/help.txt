====== Bytle Db - Help ======



===== About =====
''help'' on a [[module|module]] or on a [[action|action]] can be found:
  * at the command line with the ''%%--help%%'' or ''%%-h%%'' [[flag|flag]]
  * or in the [[start|Bytle Db documentation]]


===== Example =====
''help'' on the [[table:load|load of a table]].

<unit>
<code dos table_load_help.cmd>
db table load --help
</code>
<console>
Load a file into a database.

Syntax:

	db table load [options|flags] <File.csv>

where:

	Connection
		* --db.name <value>             defines the database name. 
A database connection is defined through its name. To know more about, see the command `db database --help`

	Args:
		* File.csv                      A CSV file

The last argument is a CSV file
</console>
</unit>
