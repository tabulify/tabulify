====== Bytle Db - Relational Table ======

===== About =====
A ''relational table'' is a [[..resource|data resource]] located in a [[relation:relation|relational database]].



===== URI =====
A [[..:data_uri|data URI]] permits to identify the table

<code bash>
@database[/schema]/table
# or
@database/table
</code>
where:
  * ''database'' is a [[database:database#name|database name]]
  * ''schema'' is a [[schema:start#name|relational schema name]]. It defaults to the ''database current schema''
