====== Bytle Db - Table ======


===== About =====
The ''table'' [[:module|module]] helps you to perform operations against one or more tables.

===== Operations =====
  * [[load|load]] -            load data into a table from a local file
  * [[download|download]] -         download a table into a local file
  * [[transfer|transfer]] -         transfer a table from a database to another
  * [[fill|fill]]              fill a table with generated data
  * [[list|list]]              list tables
  * [[drop|drop]]              drop table(s)
  * [[count|count]]             count the number of tables
  * [[show|show]]              show the table structures

===== Syntax =====

<unit>
<code dos>
db table --help
</code>
<console>
operations on one or several tables

Syntax:

	db table <command>

where:

	Command (is one of):
		* load                          load data into a table from a file
		* download                      download a table into a file
		* transfer                      transfer a table from a database to another
		* fill                          fill a table with generated data
		* list                          list tables
		* drop                          drop table(s)
		* count                         count the number of tables
		* describe                      show the table structures
		* show                          show the data of the table
</console>
</unit>
