====== Bytle Db - Relational Table ======

===== About =====
A ''relational table'' is a [[..resource|data resource]] located in a [[relation:relation|relational database]].



===== Uri =====
A [[..:data_uri|data URI]] permits to locate one or more tables

<code bash>
@database[/schema]/table
# or
@database/table
</code>
where:
  * ''database'' is a [[database:database#name|database name]] (or a [[wp>Glob_(programming)|glob pattern]])
  * ''schema'' is a [[schema:start#name|relational schema name]] (or a [[wp>Glob_(programming)|glob pattern]]). If not present, it will default to the ''database connection schema''
  * ''table'' is a ''table name'' (or a [[wp>Glob_(programming)|glob pattern]])
