# Tower App

## About

This application is called `tower` because it's one
app that serves several.

See the [doc](doc/README.md)

If an api is called `tower`, it's the private api served
by the `tower` app.

## Combo

It serves all eraldy apps:
* `api.eraldy.com/combo/_private`: the private api that is used by first-party developer
* `api.eraldy.com/combo`: the public api that is used by third-party developer
* `combo.eraldy.com`: the combo app front end
* `member.combostrap.com`: the member / auth api (login, registration, ..)
